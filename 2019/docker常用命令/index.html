Docker常用命令

Written by 晗月 with ♥ on 26 August 2019 in 码间小记 1 min

docker

  • 查看正在运行的容器: docker ps
  • 查看所有容器: docker ps -a
  • 启动 : docker start ***
  • 停止 : docker stop ***
  • 重启 : docker restart ***
  • 删除 : docker stop ***&& docker rm ***
  • 更新 镜像: docker pull ***

docker-compose

version: "3"
services:
  neteaseCloudMusic:
    image: binaryify/netease_cloud_music_api
    container_name: netease_cloud_music_api
    restart: always 
    ports:
      - "3000:3000"
  • 部署 : docker-compose up -d
  • 启动 : docker-compose start ***
  • 停止 : docker-compose stop ***
  • 重启 : docker-compose restart ***
  • 删除 : docker stop && docker rm ***
  • 更新 : docker-compose pull && docker-compose up -d
