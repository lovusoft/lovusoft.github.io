Angular手写音量滑动条

Written by 晗月 with ♥ on 2 September 2019 in 码间小记 1 min

最终效果

HTML:

<input type="range" min="0" max="100" value="30" [(ngModel)]="sound" [ngStyle]="{'background-size': process}" />
<!--ngModel绑定当前数值到sound-->
<!--ngStyle从ts获取当前滚动进度，设置对应的背景颜色-->

SCSS:

  .sound {
      white-space: nowrap;

      input {
        margin-left: 6px;

        &[type="range"] {
          appearance: none;		//清除默认样式
          background: linear-gradient(#d33a31, #d33a31) no-repeat, #f5f5f5;
          //linear-gradient设置滑动进度前后颜色
          height: 2px;
          position: relative;
          bottom: 8.5px;
          border: none;
            
		  //设置滚动条上按钮样式
          &::-webkit-slider-thumb {
            appearance: none;
            cursor: pointer;
            position: relative;
            box-sizing: border-box;
            width: 10px;
            height: 10px;
            border-radius: 50%;
            background: #d33a31;
          }

        }
      }
    }

Typescript:

sound = 20; 
//与Vue计算属性相似
get process() {
    return this.sound + '% 100%';
  }
